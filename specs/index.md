# Physical AI & Humanoid Robotics - Specifications

Welcome to the specifications directory for the Physical AI & Humanoid Robotics textbook project. This collection of documents outlines the comprehensive requirements, architecture, and design decisions for the AI-native educational platform.

## Specification Documents

### [00-project-overview.md](00-project-overview.md) - Project Overview
High-level summary of the textbook project, including goals to teach Physical AI, embodied intelligence, and humanoid robotics. Defines the target audience of students transitioning from digital AI to physical robotics, with key outcomes focused on building simulated autonomous humanoids.

### [01-core-features.md](01-core-features.md) - Core Features
Detailed description of the implemented core features including the Docusaurus-based interactive textbook, complete course content coverage, embedded RAG chatbot with global and selected-text context support, and GitHub Pages deployment.

### [02-bonus-features.md](02-bonus-features.md) - Bonus Features Implemented
Specifications for additional features including Better Auth authentication, user profile collection, per-chapter content personalization, Urdu translation toggle, and reusable intelligence via subagents.

### [03-technical-architecture.md](03-technical-architecture.md) - Technical Architecture
Technical architecture overview covering the Docusaurus v3 frontend, FastAPI backend, Qdrant Cloud vector database, Neon Serverless Postgres, and deployment strategy with content ingestion flow.

### [04-content-structure.md](04-content-structure.md) - Content Structure & Navigation
Detailed content structure with sidebar hierarchy matching the course, chapter breakdowns for all modules and weeks, and specifications for content elements like code blocks, tables, diagrams, and interactive elements.

### [05-user-experience.md](05-user-experience.md) - User Experience & Interactivity
User experience specifications for the RAG chatbot widget, selected text context support, personalization logic, Urdu translation functionality, and mobile-responsive design.

### [06-deployment-and-maintenance.md](06-deployment-and-maintenance.md) - Deployment & Maintenance
Deployment configuration for GitHub Pages and backend hosting, CI/CD pipeline specifications, content management processes, and future extensibility considerations.
