# Core Features

## Docusaurus-based Interactive Textbook
- Complete interactive textbook built with Docusaurus v3
- Full course content covering all modules and weekly breakdowns
- Responsive design for multiple device types
- Search functionality for easy content discovery

## Complete Course Coverage
- Module 1: ROS 2 (Robot Operating System 2)
- Module 2: Gazebo/Unity simulation environments
- Module 3: NVIDIA Isaac Sim platform
- Module 4: Vision-Language-Action (VLA) models
- Weekly breakdown covering Weeks 1-13
- Hardware requirements guide
- Capstone project with implementation guidelines
- Learning outcomes and assessments

## Embedded RAG Chatbot
- Contextual AI assistance using Retrieval-Augmented Generation
- Supports global questions about the entire textbook
- Selected text context support for specific content queries
- FastAPI backend for processing requests
- Integration with OpenAI for natural language understanding
- Qdrant Cloud vector database for content indexing
- Neon Serverless Postgres for user data and metadata

## GitHub Pages Deployment
- Static site hosting on GitHub Pages
- Optimized loading performance
- Mobile-responsive design
- SEO-friendly structure
- CDN delivery for global access
